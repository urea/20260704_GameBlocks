# GameBlocks Module Copy Area

Copy selected modules from `../../gameblocks/modules/` into this folder when a prototype starts using them.

Preserve the upstream relative path. For example:

```text
gameblocks/modules/math/WorldBasis.js
```

should be copied to:

```text
src/gameblocks/modules/math/WorldBasis.js
```

Record every copied or adapted module in `../../gameblocks_usage.md`.
